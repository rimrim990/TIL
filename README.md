**공부 시간**
- 우아한 형제들 CTO 님께서 개발자의 성장 가능성은 다음과 같은 공식으로 측정할 수 있다고 말씀하셨다
  - **성장 가능성** = 재능 x 학습 방법 x 투자한 시간 x 환경
- 이 중에서 내가 가장 쉽게 향상시킬 수 있는 요소는 **투자한 시간**이다
- 따라서 매주 50시간 이상을 학습에 투자하는 것을 목표로 한다
  - 학습 시간은 열품타로 측정
